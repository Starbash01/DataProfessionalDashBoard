# DataProfessionalDashBoard
Power BI Data Professional Dashboard analyzing survey responses from data professionals. The dashboard explores salary trends, job roles, programming language preferences, work-life balance, job satisfaction, and career demographics through interactive visualizations, providing insights into industry trends and workforce experiences.
